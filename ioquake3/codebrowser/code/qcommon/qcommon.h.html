484
485cvar_t variables are used to hold scalar or string variables that can be changed
486or displayed at the console or prog code as well as accessed directly
487in C code.
488
489The user can access cvars from the console in three ways:
490r_draworder prints the current value
491r_draworder 0 sets the current value to 0
492set r_draworder 0 as above, but creates the cvar if not present
493

495interface from being ambiguous.
496

498modules of the program.
499
500*/
